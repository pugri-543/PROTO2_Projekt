# PROTO2 Project

Welcome to our PROTO2 Project!

We are two students from the Lycée des Arts et Métiers in Luxembourg, studying TPIF (IT Technician).
This project was created as part of our PROTO2 course.

Our project involves using a Raspberry Pi 5 running Node-RED and a Minecraft server, alongside a Discord 
bot and a DigiLab setup.
To accomplish this, we use Node-RED to establish connections between the Discord bot, the Minecraft server,
and the DigiLab through various associated nodes.
